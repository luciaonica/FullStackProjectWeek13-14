Welcome to the Client Management App

About this application

Navigate through the navigation bar

  • Features a navigation bar at the top that allows for you to move between each component

Home

  • Returns to this page.

Projects

  • Look and manage your projects.
  • You can cancel them by pressing the cancel button.

Add project

  • Add a project based on name.
  • Can manage it through the projects tab.

Clients

  • Only visible to developer users.
  • Allows for you to see the registered clients and their projects when clicking the client name.
  • Allows to mark them as completed.
  • Allows for you to update the client information.
  • Allows to add projects to each user.

Add client information

  • Insert your personal information in the add client information tab to start using the app.
  • Personal information includes your name, email, address, and an agreement PDF file.
  • Can only do it once, option disappears after submitting.
  • Other nav bar items appear after submitting this form.

Update account information

  • Allows for you to update your password.
  • Immediately logs out after updating the password.

Create dev account

  • Only visible to developer users.
  • Allows for you to create more developer accounts.
